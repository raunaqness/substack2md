# substack2md

Paste a free Substack article URL and get clean Markdown back — with a single click to copy or download as a `.md` file.

**[substack2md.vercel.app](https://substack2md.vercel.app)**

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
